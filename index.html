왜 만드는가

😵 문제

슬랙 88개 메시지를 처음부터 읽어야 현황 파악. 노션 트래커 2/13 멈춤.

🎯 목표

당장 돌아가는 마케팅 업무를 일목요연하게. 무겁게 만들지 않음.

📏 범위

마케팅만. 다른 분야 절대 섞지 않음.

업무 카테고리

  • 콘텐츠 인스타 숏폼, 유튜브, 블로그, 크리에이티브
  • 광고 Meta, 네이버, 성과 분석, UTM/픽셀
  • 웹 상세페이지, 체험페이지, GA4
  • 이벤트 스포엑스, 프리오더, 체험 신청
  • 전략 브랜딩, 채널, 프로필

정보 계층 (4층 구조)

1층 — 대시보드

전체 업무 한눈에 (칸반/리스트)

2층 — 카드 상세

각 업무의 현재 상태 요약

3층 — 히스토리

고릴이 정리한 진행 경과

4층 — 레퍼런스

"왜 이렇게 정리했지?" → 원본 추적

평소엔 1~2층만. 의문 시 3~4층까지.

데이터 흐름

📥 입력

텔레그램 토픽 (실시간) + 슬랙 (30분 sync) + 대시보드 직접 입력

🤖 처리

AI 감지 → 담당자 컨펌 → 등록. 자동 확정 안 함.

📊 출력

웹 대시보드 + 텔레그램 알림 (매일 아침/마감 임박/주간)

확정된 결정사항 (3/5)

✅ DB: JSON (가볍게)
✅ 당장 돌아가는 업무 확인용 (무겁게 X)
✅ 텔레그램 자연어 + 대시보드 직접 수정 가능
✅ 감지 → 담당자 컨펌 → 등록
✅ 마케팅만. 다른 업무 절대 섞지 않음
✅ 담당자별 뷰 분리 안 함 (표시만)
✅ 텔레그램 토픽 기반 관리
✅ 별도 웹페이지 (mkt-dash.wespion.com)
✅ 블로커 표시 + 최근순 정렬 + 업데이트 경과
