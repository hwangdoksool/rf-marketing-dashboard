1. 왜 만드는가

😵 현재 문제

슬랙 메시지 88개를 처음부터 읽어야 현황 파악. 노션 트래커는 2/13에 멈춤. 누가 뭘 하는지 물어봐야 앎.

🎯 목표 상태

대시보드 한 번 열면 전체 업무 현황 즉시 파악. 우선순위·마감·담당 한눈에. AI가 자동 수집·알림.

황성우 (총괄/의사결정)
형은님 (마케팅 실행)

10. 확정된 결정사항

✅
태스크 DB → JSON
가볍게. Worker 또는 로컬 저장.
✅
당장 돌아가는 업무 확인용
무겁게 X. 실용적으로.
✅
텔레그램 자연어 기본 + 대시보드 직접 수정 가능
형은님도 직접 수정 권한 있음.
✅
감지 → 담당자 컨펌 → 등록
AI 자동 확정 안 함. 귀찮으면 나중에 재검토.
✅
마케팅만. 다른 업무 절대 섞지 않음.
헷갈려서 더 안 좋아짐.
